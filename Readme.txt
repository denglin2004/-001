     AIN1  :     PD2
     AIN2  :     PD1
     PWMA  :     PA0

     BIN1  :     PD3
     BIN2  :     PD4
     PWMB  :     PA2

     TB6612FNG驱动芯片2  
     CIN1  :     PE12
     CIN2  :     PE11
     PWMC  :     PA2

     DIN1  :     PE13
     DIN2  :     PE14
     PWMD  :     PA3

 * 2 电机编码器接口
    左前轮  TIM2  :PB3 PA15
    左后轮  TIM3  :PB4 PB5
    右前轮  TIM4  :PD12 PD13
    右后轮  TIM8  :PC6 PC7

 * 3 定时器1:
    TIM1 中断计数，用于时间切片调度任务(如PID任务，编码器计数，RS485接受数据处理，CAN数据接受处理，USART数据处理等等)

 * 4 RS485接口:
  TXD :USART1_TX -->PA9
  RXD :USART1_TX -->PA10

 * 5 CAN接口:
  TXD : CAN1_TX  -->PA12
  RXD : CAN1_RX  -->PA11

 * 6 OLED接口:
  GND :  -->PE3
  VCC :  -->PE4
  SCL :  -->PE5
  SDA :  -->PE6
